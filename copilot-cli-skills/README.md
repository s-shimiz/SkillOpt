# Copilot CLI 用 SkillOpt スキル集

このフォルダは、SkillOpt / SkillOpt-Sleep の仕組みを **GitHub Copilot CLI のスキル（Skill）** として
すぐ使える形にまとめた成果物です。技術的に初心者の方でも導入できるように手順書を同梱しています。

> 最終更新日: 2026-06-26

## このフォルダに入っているもの

| 種類 | ファイル / フォルダ | 説明 |
|---|---|---|
| 作業報告書 | [`作業報告書.md`](作業報告書.md) | 何を・なぜ・どう作ったかの記録（背景・設計・検証結果） |
| 手順書 | [`セットアップガイド.md`](セットアップガイド.md) | インストールと各スキルの使い方（初心者向け） |
| スキル本体 | `skillopt-sleep/` 他 5 フォルダ | Copilot CLI が読み込むスキル一式 |

## 5 つのスキル

| スキル | ひとことで言うと | 自己完結 |
|---|---|---|
| [`skillopt-sleep`](skillopt-sleep/) | 過去の作業から学んでスキルを自動改善（夜間バッチ向き） | ✅ |
| [`skillopt-now`](skillopt-now/) | 今すぐ 1 回だけスキルを最適化 | ✅ |
| [`skillopt-eval`](skillopt-eval/) | スキルの品質をスコアで測定 | ✅ / ⚠️ |
| [`skillopt-new-skill`](skillopt-new-skill/) | 新しいスキルをゼロから作成 | ✅ |
| [`skillopt-train`](skillopt-train/) | ベンチマークでフル学習（研究用） | ⚠️ |

- **✅ 自己完結**: 追加インストール不要（エンジンを `skillopt-sleep/vendor/` に同梱）。
- **⚠️**: 一部機能のみ、SkillOpt 本体リポジトリと依存パッケージが必要。

## まず動かしてみる（30 秒）

```powershell
# 同梱エンジンが正しく動くことを証明（API キー不要・何も変更しない）
& ".\skillopt-sleep\run.ps1" --proof
```

末尾に `PASS:` と表示されれば成功です。詳しくは [`セットアップガイド.md`](セットアップガイド.md) を参照してください。

## 用語ミニ辞典

- **スキル（Skill）**: Copilot CLI に機能を追加する仕組み。`SKILL.md` に手順を書く。
- **SKILL.md**: スキルの本体。Copilot がいつ・どう使うかを記したテキスト。
- **SkillOpt**: スキル文書を「学習対象」とみなし、検証付きで自動改善する手法。
- **held-out（ホールドアウト）スコア**: 学習に使わず取り置いた問題での正答率。改善の良し悪しを公平に測る指標。
- **vendor（同梱）**: 外部リポジトリに頼らず、必要なコードをこのフォルダに同梱すること。
